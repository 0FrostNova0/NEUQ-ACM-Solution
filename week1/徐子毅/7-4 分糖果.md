# 7-4 分糖果

> 现在有n个糖果和一群小朋友，第一个小朋友拥有这n个糖果，他现在有两种选择①分给第二个小朋友x个，x必须是n的约数，且x<n②全部自己留着，第二个小朋友同样也是要么分自己拥有糖果数的约数个给下一个小朋友，要么全留着，以此类推。现在给出第一个小朋友的糖果数n，询问有多少种分法。

这里我用的是递归，定义一个函数表示n个的分发，因为分发都是一样的解法。

```C++
#include <iostream>
using namespace std;
int func(int n){
	int sum=0;
    if(n==1) return 1;
    else{
        sum=0;
        for(int i=1;i<n;i++){
            if(n%i==0) sum=sum+func(i);
        }
    }
    return sum;
}
int main(){
    int N;
    cin >> N;
    cout << func(N)*2 << endl;
    return 0;
}
```
